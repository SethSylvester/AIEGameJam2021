# 2021-Gamejam-AIE
## Summary:

Created as part of an 72 hour gamejam hosted by Academy of Interaction Entertainment, Lafayette Louisiana campus. You play as a knife trying to reforge all of its pieces through various minigames.

## Controls:

* Left click - Interact
* W - Moving Forward
* S - Moving Backwards
